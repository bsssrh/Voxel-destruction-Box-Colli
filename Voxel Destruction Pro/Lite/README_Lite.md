# Lite Voxel Scripts (Минимальный набор)

Этот набор — максимально простой «клон» основной системы, но без расширенных настроек.
Используйте **только** эти новые скрипты для упрощённого пайплайна.

## Что входит
- `LiteVoxelManager` — один менеджер с базовыми ссылками на настройки.
- `LiteVoxLoader` — загрузка `.vox` + подготовка MeshCollider.
- `LiteLaserDestroy` — простой «лазер» разрушения.
- `LiteExplosion` — простой взрыв.

## Быстрый старт
1. **Добавьте менеджер**
   - Создайте пустой объект `LiteVoxelManager`.
   - Проставьте ссылки на `MeshSettings`, `IsoSettings`, `DynSettings` и материал.

2. **Создайте воксельный объект**
   - Добавьте `DynamicVoxelObj` на объект.
   - Добавьте `LiteVoxLoader` и укажите `voxFile` (Editor) или `modelPath`.
   - `LiteVoxLoader` сам добавит `VoxFileDataProvider` и `MeshCollider`.

3. **Разрушение лазером**
   - Добавьте `LiteLaserDestroy` на камеру или любой объект.
   - Укажите `rayCamera` (если используете курсор) или `rayOrigin`.

4. **Взрыв**
   - Добавьте `LiteExplosion` на объект.
   - Установите `delay` и `radius`.

## Ограничения (намеренно)
- Минимум параметров в инспекторе.
- Нет фильтров материалов, нет окраски, нет тонкой настройки разрушения.
- Всегда используется **MeshCollider**.

Если нужно «как в полной версии», используйте старые скрипты. Этот набор — только для простых демонстраций.
